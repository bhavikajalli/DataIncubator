# DataIncubator
The dataset consists of tweets collected using Tweepy from February 25th to March 31st 2018. 

Each tweet is in the form of a dictionary with the fields
-	screen name, 
-	re-tweet_count, 
-	replied_to, 
-  text 
-	Location
-	Hashtags_in_the_tweet
-	created _at
-	Mentions_in_the-tweet
-	Name and
-	User_ID
